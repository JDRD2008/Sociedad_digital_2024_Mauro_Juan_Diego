Software

Definición

El software es el conjunto de construcciones, datos y programas que le indican a un ordenador qué hacer. El software es intangible, a diferencia del hardware, que son los componentes físicos. El software ejecuta el hardware para realizar funciones y operaciones. Se clasifica en las siguientes categorías:

Categorías de Software

  • Software de Sistema:
    • Sistema Operativo (OS)
    • Controladores de dispositivos (drivers)
  • Software de Aplicación:
    • Programas de Productividad
    • Aplicaciones Móviles
  • Software de Desarrollo:
    • Lenguajes de Programación
    • Entornos de Desarrollo Integrado (IDE)
  • Software de Utilidad:
    • Herramientas de Mantenimiento
    • Compresores de Archivos
